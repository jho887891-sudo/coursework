# 公开数据说明

本目录全部是为教学构造的 fixture，不是现行法律、政策或真实交易数据。用途是保证每位学生在离线环境使用相同输入，公平比较 Workflow、Agent、检索和评测设计。

- `lesson-03-fixtures/`：工具读取与权限测试；
- `lesson-04-evidence/`：证据检索、拒答、冲突和注入测试；
- `lesson-05-travel/`：规划与重规划场景；
- `final-project/`：20 条固定公开评测、10 条完整来源和 11 份教学文档；隐藏集不得放在学生目录。

当前固定规模：Lesson 3 为 12 条 Tool case，Lesson 4 为 20 条 query，Lesson 5 为 30 条 Planning task，Final Project 为 20 条 eval case。学生可以生成额外变体，但正式对比必须包含这些固定样本。
