## 规则引擎 vs AI Agent —— 分工

```
一份招标文件 200 条 clauses

规则引擎（50ms）：                    AI Agent（~30s）：
  "公告期 15 天，法定 ≥ 20"  → 画红笔    "投标人须具有相应的资质"
  "报价 600 万，预算 500 万" → 画红笔    → "相应"是模糊表述还是合理简写？
  "缺安全生产许可证"        → 画红笔    → 需要理解上下文和行业惯例
  "营业执照过期"           → 画红笔    
  "未明确资质等级"          → 画红笔    

引擎筛完：12 条确定性问题已标记           Agent 只审剩下的：5 条需要语义理解的条款
```

**规则引擎不聪明。正因为不聪明，它快、稳、可解释。**
