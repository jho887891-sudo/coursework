#M3薄弱点列表模块文档

## 一、模块名称

薄弱点列表模块

## 二、模块简介

薄弱点分析模块旨在通过深度分析用户的做题记录、AI讲题反馈及提问历史，自动识别并量化用户的知识薄弱点。模块以列表和知识图谱的形式直观展示分析结果，并据此提供精准的针对性练习推荐。

## 三、主要功能

综合多源数据，进行薄弱点计算与分析
以列表和知识图谱的形式，查询与展示薄弱点
根据薄弱点，为用户推荐针对性练习题目。
追踪薄弱点的历史变化，生成改善报告，量化学习成果。

## 四、基本工作流程

系统定时或实时采集用户的答题、错题及提问数据。
薄弱点计算引擎读取数据，运用统计算法计算每个知识点的薄弱分数和趋势。
计算结果被写入数据库，并通过API接口提供给前端及其他模块。
前端请求薄弱点数据，展示薄弱点列表和知识图谱。
用户点击薄弱点，可查看其详细分析、关联的错题，并启动针对性练习。
用户完成练习后，系统会增量更新薄弱点数据，反映最新的学习状态。

## 五、使用技术

开发工具：Visual Studio, Git, GitHub
前端： TypeScript
后端 ：Java, Python
