# M5 AI备课模块 — 需求文档

## 一、模块名称
**M5 AI备课模块**

## 二、模块简介
AI备课模块帮助用户（教师/学生/家长）快速生成教学大纲和PPT课件。用户输入知识点和教学目标，AI自动生成结构化教案，并可进一步编排为可编辑的PPT，支持导出和推送给即时讲课（M4）使用。

## 三、主要实现功能

| 功能 | 说明 |
|------|------|
| **AI备课内容生成** | 输入知识点+教学目标+课时数 → AI生成教学大纲 + 逐课时详细教案 |
| **教案编辑** | 用户可修改大纲、课时内容、增删课时，支持富文本编辑 |
| **AI PPT编排** | 基于教案自动编排幻灯片（分页、图文布局） |
| **PPT预览编辑** | 缩略图导航、Canvas预览、拖拽排序、增删页面 |
| **PPT风格自定义** | 配色/字体/布局自定义，支持保存为模板复用 |
| **PPTX导出** | 导出为标准PPTX文件下载 |
| **M4联动** | 备课内容一键推送给即时讲课使用 |

## 四、基本工作流程

```
用户输入(知识点 + 教学目标 + 课时数)
    → AI生成教学大纲(流式) → 逐课时生成教案(流式)
        → 用户编辑确认教案
            → AI编排PPT结构
                → 用户预览/编辑PPT
                    → 导出PPTX / 推送给M4讲课
```

## 五、使用技术

| 层 | 技术 |
|----|------|
| **前端** | React + TypeScript |
| **业务后端** | Java Spring Boot + MyBatis-Plus |
| **AI微服务** | Python FastAPI |
| **数据层** | MySQL + Redis |
| **PPT导出** | Apache POI (Java) |

- **Java负责**：备课CRUD、模板管理、PPTX导出、调用Python服务
- **Python负责**：AI备课内容生成、PPT动态编排、Prompt工程
